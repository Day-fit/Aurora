# Aurora
Aurora is a web application for generating resumes (CVs) with optional enhancement powered by AI (OpenAI). Designed for simplicity and efficiency, Aurora allows users to create, edit, and export professional CVs effortlessly.

[![build-docker.yml](https://github.com/Day-fit/Aurora/actions/workflows/build-docker.yml/badge.svg)](https://github.com/Day-fit/Aurora/actions/workflows/build-docker.yml)

## Features

CV Generation: Create resumes quickly through a simple web interface.

AI Enhancement: Improve your CV content using OpenAI's AI capabilities.

PDF Export: Download your CV in PDF format.

Edit & Save: Save your CVs for later editing and updates.

## Technology Stack

Language: Kotlin

Architecture: Microservices

## Installation & Hosting

Aurora is primarily intended as a hosted web application and is not designed for self-hosting.
